# Внедрение QR-CODE в жизнь школы
Создание телеграм-бота на языке python, для обеспечения школьникам пропуска и выхода из учебного учреждения.

Бот доступен в телеграм по названию @qrcode_962_bot или [по ссылке](https://t.me/qrcode_962_bot)

## Как запустить
Для работы кода необходимо установить библиотеки telebot и qrcode с помощью команд в терминале:
```bash
pip install telebot
pip install qrcode
```
Токен бота есть у разработчиков приложения.
